## alias命令
git命令起别名
```
#设置checkout的别名为co
$ git config --global alias.co checkout
#设置branch的别名为br
$ git config --global alias.br branch
#设置status的别名为st
$ git config --global alias.st status
#设置commit的别名为ci
$ git config --global alias.ci commit
#设置log--oneline一行显示的别名为lol
$ git config --global alias.lol "log--oneline"
```
